hinawa_fw_req_transaction_async

Execute request subaction of transactions to the given node according to given code. When the response subaction arrives and read the contents, HinawaFwReq::responded signal handler is called as long as event dispatcher runs.

Available since: 2.1.

hinawa_fw_req_transaction_sync

Execute request subaction of transaction to the given node according to given code, then wait for response subaction within the given timeout. The HinawaFwReq:timeout property of instance is ignored.

Available since: 2.1.
