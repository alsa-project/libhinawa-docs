Description

A HinawaFwResp responds requests from any units.

This class is an application of Linux FireWire subsystem. All of operations utilize ioctl(2) with subsystem specific request commands.

The “requested” signal

HinawaFwRcode
user_function (HinawaFwResp *self,
               HinawaFwTcode tcode,

When any node transfers requests to the range of address to which this object listening. The “requested” signal handler can get data frame by a call of hinawa_fw_resp_get_req_frame() and set data frame by a call of hinawa_fw_resp_set_resp_frame(), then returns rcode.
